# Dwarf MMA App

## To run locally
 - ```npm install```
 - ```npm start```
 - ```start Dwarf MMA server```

# Run Dwarf MMA server
- pull the image:   ```docker pull rushkappa/dwarf-mma-server:latest``` 
- start the container:  ```docker run -d -p 2137:2137 rushkappa/dwarf-mma-server:latest```
